# 🚖 Taxi Data Analysis using Python

> A complete Exploratory Data Analysis (EDA) project on a Taxi Booking dataset using Python, Pandas, NumPy, and Matplotlib.

---

## 📌 Project Overview

The objective of this project is to analyze taxi booking data and extract meaningful business insights through data cleaning, aggregation, and visualization.

This project demonstrates practical Data Analytics skills such as:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insight Generation

---

## 📂 Dataset Features

The dataset includes information such as:

- Trip Date
- Pickup Location
- Vehicle Type
- Distance (KM)
- Fare Amount
- Customer Rating

# 📊 Exploratory Data Analysis (EDA)

The following analyses were performed:

### 📈 Vehicle-wise Revenue
Analyzed the total revenue generated by each vehicle type.

### 🌍 City-wise Revenue
Compared revenue generated across different pickup cities.

### 📅 Monthly Revenue Trend
Observed monthly revenue trends throughout the year.

### 🚕 Distance vs Fare Analysis
Studied the relationship between trip distance and fare amount.

### 💰 Revenue Share by City
Calculated each city's contribution to total revenue.

### ⭐ Vehicle-wise Average Rating
Compared average customer ratings across vehicle categories.

### 📉 Fare Distribution
Visualized the distribution of fare amounts.

# 💡 Key Insights

- Mini vehicles generated the highest overall revenue.
- Revenue contribution from all vehicle categories is fairly balanced.
- Noida generated the highest city-wise revenue.
- Monthly revenue remained stable with a noticeable peak in December.
- Fare generally increases with trip distance, showing a positive relationship.
- Most trips fall within the low to medium fare range.
- Customer ratings are consistent across all vehicle types, indicating overall customer satisfaction.
